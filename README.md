# ClipboardTextTyper
## Описание проекта
Это программа, которая позволяет обойти блокировку вставки текста в приложениях.
Программа берет текст из буфера обмена, затем программно нажимает клавиши и печает текст в активном приложении.
### Установка
Для установки программы необходимо скачать из папки **```Installer/Release```** файл **```Setup.msi```**, или воспользоваться [ссылкой](Installer/Release/Setup.msi?raw=true).
При открытии файла запустится стандартный установщик, нужно будет проследовать по всем шагам. После установки программма будет отображаться в списке установленных программ.
### Удаление
Чтобы удалить программу, воспользоваться любым деинсталлятором. можно в меню **"Пуск"** нажать **правой кнопкой по иконке приложения**, затем **"Удалить"** или воспользоваться любым деинсталлятором.
Программа удаляется полностью, не оставляя после себя ничего.
### Использование
Чтобы запустить программу, можно воспользоваться ярлыком на рабочем столе или найти ее в меню пуск.

В программе есть:
- [x] Графический интерфейс
- [x] Можно включить или выключить задержку при печати каждого символа
- [x] Можно указать задержку в милисекундах
- [x] Можно указать символы, которые нужно убрать при печати.
- [x] Автосохрание и автозагрузка настроек
   
   
##### Таблица горячих клавиш:
Горячие клавиши | Действие
----------------|---------
```Shift``` + ```F2``` | Начать печать
```Shift``` + ```F4``` | Завершить печать
```Shift``` + ```F11```| Остановить печать
```Shift``` + ```F12```| Продолжить печать

**Пока зажата** клавиша **```Shift```**, **печать приостанавливается**, программа ожидает выбора функции от пользователя.
При переключении настроек, они **автоматически сохраняются** в специальный файл ```Settings.json```, поэтому **при перезапуске** программы, **все настройки возвращаются**.

**Все сочетания клавиш еще раз написаны в программе**

Для игнорирования определенных символов есть специальное поле для ввода, можно вводить как обычные символы, так и **специальные управляющие символы**(таблица приведена ниже). 
**При вводе одного или нескольких пробелов**, **они будут считаться как группы символов**.
Пример: если ввести два пробела подряд, из текста пропадут все двойные пробелы.
##### Таблица специальных символов:
Символ | Описание
-------|--------
\n | переход на новую строку
\t | горизонтальная табуляция
